Installation
============

Basic Installation
------------------

Install PrefGraph using pip:

.. code-block:: bash

   pip install prefgraph

Optional Dependencies
---------------------

For visualization support (matplotlib):

.. code-block:: bash

   pip install prefgraph[viz]

For Jupyter notebooks with full data science stack:

.. code-block:: bash

   pip install prefgraph[notebooks]

For development (testing, linting):

.. code-block:: bash

   pip install prefgraph[dev]

For everything:

.. code-block:: bash

   pip install prefgraph[all]

Requirements
------------

- Python 3.10 or higher
- NumPy >= 1.24.0
- SciPy >= 1.10.0
- NetworkX >= 3.0
- Numba >= 0.58.0
